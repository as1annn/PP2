195728. Newbie.

The element of the array is called a $newbie$, if there was no same element in the array before.

For every element of the array, print ``\t{YES}'' if it is newbie, and ``\t{NO}'' otherwise.

The first line of input contains an integer $n$ $(1 \le n \le 10^4)$. Next line contains $n$ integers - elements of the array $a$ $(1 \le a_i \le 10^6)$.

For every element of the array, print ``\t{YES}'' if it is newbie, and ``\t{NO}'' otherwise.
